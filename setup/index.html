Configurer et diagnostiquer la connexion Grist pour les widgets standalone (iPhone)

Ou trouver ma cle API Grist

  1. Va sur https://grist.playwubo.com
  2. Connecte-toi si ce n'est pas fait
  3. Clique ton avatar en haut a droite
  4. Clique Profile Settings
  5. Descends jusqu'a la section API
  6. Si aucune cle : clique Create
  7. Clique sur la cle affichee pour la copier. Elle ressemble a abc123def456... (plus de 24 caracteres alphanumeriques)

Ne confonds pas avec : User ID, Doc ID, ou ton email. C'est la LONGUE chaine aleatoire.
